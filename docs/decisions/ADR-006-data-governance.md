# ADR-006: Data Governance & Privacy Framework (v6.2)

**Status:** ACCEPTED  
**Date:** 2026-03-01  
**Context:** PDPL (Saudi), GDPR compliance requires systematic data governance.  
**Decision:** 5-level data classification (L1-L5), centralized Data Classification Registry in K8, PII tracking and masking, Right to Erasure support, consent management per tenant.  
**Consequences:** Full regulatory compliance readiness. Automated retention enforcement. Privacy-by-design.  
**Rules:** DGP-001 through DGP-026, FP-050 through FP-054
