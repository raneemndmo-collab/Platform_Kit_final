# ADR-012-I18N-RTL

## Internationalization and RTL Support

**Status:** Accepted
**Date:** 2026-03-01
**Deciders:** Architecture Review Board

## Context
Platform must support Arabic-first with full RTL and bilingual operations

## Decision
Dedicated i18n service with locale-aware formatting for dates, numbers, currencies

## Consequences
All UI components must respect locale direction.
